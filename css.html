О языке CSS

CSS (Cascading Style Sheets) используется для стилизации и внешнего оформления веб-документов, созданных с использованием HTML. Он позволяет задавать цвета, шрифты, отступы и другие стили, делая страницы более привлекательными и удобочитаемыми.

Пример использования CSS для стилизации текста:

h1 {
  color: orange;
  text-align: center;
}

CSS работает на основе каскадности и наследования стилей. Это позволяет разработчикам легко управлять внешним видом элементов на странице и создавать унифицированный и современный дизайн.

С помощью CSS можно также реализовать адаптивный дизайн, который обеспечивает корректное отображение веб-сайта на различных устройствах и экранах.


Пример страницы с использованием CSS:

<h1 style="color: green; text-align: center">Этот заголовок зеленый и по центру</h1>

<p style="color:red">Этот абзац красный!</p>
